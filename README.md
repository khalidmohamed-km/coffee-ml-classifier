# Coffee ML Classifier ☕

This is a simple machine learning project using Python and Decision Tree.

The model predicts the type of coffee based on:
- coffee amount
- milk amount
- foam amount

Technologies used:
- Python
- numpy
- scikit-learn

## How to use

1. Run `coffee_model.py`
2. Enter your coffee, milk, and foam amounts
3. See the predicted coffee type!
